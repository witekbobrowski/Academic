# Zestaw 5

## Generics

This time we got to implement a Merge sort algorithm, and code our own Color enum with ColorMixer, which can mix elements of Color enum and return the RGB values.

## What's included

```
generics/
├── ColorMixer.java
├── MergeSort.java
└── Tester.java
```

## Usage

Compile and run

```
javac generics/*.java
java generics.Tester
```
